This folder contains dairy_sires.csv, extracted from the openly-licensed
(CC BY 4.0) Mendeley dataset:

  Sarah, P., Tasrifin, D. S., Indrijani, H., & Ruswandi, D. (2024).
  "Dataset for performance of superior dairy cattle sires."
  Mendeley Data, v2. https://doi.org/10.17632/2sm93h8t7y.2

It was built by combining the four lactation sheets of the source workbook into
one table (2,179 cow records after removing summary-statistic rows). See
src/download_data.py for provenance and src/data.py for the cleaning logic.

Because the source is CC BY 4.0, you may keep this file in the repo as long as
you cite the original authors (see the main README).
